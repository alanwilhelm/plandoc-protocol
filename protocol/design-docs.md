# Design Docs

Design docs are architectural blueprint anchors.

They define durable meaning:

- system structure
- contracts and boundaries
- ownership
- vocabulary
- invariants
- accepted options
- rejected options
- proof and acceptance meaning

Plandocs execute bounded work against design docs. A plandoc may add tactical
detail, but it must not invent, weaken, or contradict durable design meaning.

If a plandoc needs meaning missing from its design anchor, refine or supersede
the design doc first.
